# Shire Monorepo

Shire is organized as an npm workspaces + Turborepo monorepo.

## Workspace layout
- `apps/web` - main product web app
- `apps/agent` - background agent service powered by Mastra
- `contracts` - Solidity workspace

## Root commands
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run typecheck`

## Product Direction
- Hiring marketplace powered by AI workflows
- Settlement and staking on Celo using stablecoins
