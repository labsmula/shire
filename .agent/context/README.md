# Context

This folder contains the primary context that should be read first.

## Priority order
1. `architecture.md`
2. More specific feature documents
3. `agent/README.md` for agent workflow context
4. `auth/README.md` for auth-specific context
5. `schemas/README.md` for shared validation schemas
6. `onchain/README.md` for contract and sync context
7. `process.md` for tool, skill, and MCP rules
8. The latest decision notes in `../decisions/`

## Working principles
- Do not guess beyond what the documents say.
- If a detail is not written yet, record it as an open question.
- If new context conflicts with old context, follow the most explicit and most recent source.
